<< 当imageフォルダは各スクリプト中で使用する画像を保存するフォルダである >>
ルール：
  ・imageフォルダ内に各スクリプト名と等しいフォルダ(拡張子は除く)を作成し、スクリプト中で使用する画像は対応するフォルダに格納する
    例: detail_window.py -> "detail_window"という名称のフォルダを作成し、そこにdetail_window.py中で使用する画像を保存する

  ・対応するフォルダ外の画像にはスクリプト内でアクセスしない

  ・自身が担当するスクリプトのフォルダ以外のフォルダ中の画像の追加・削除を行わない

  ・imageフォルダ中の各フォルダに保存するのは画像のみとする
